  • width Height : ✅ Bisa ditentukan
  • Margin Padding : ✅ Bisa ditentukan
  • Arah : ✅ Bisa ditentukan
    • Information

    1. flex-direction: dapat mengontrol arah penempatan elemen

    • flex-direction: row; elemen ditempatkan secara horizontal, dari kiri ke kanan (nilai default)
    • flex-direction: row-reverse; elemen ditempatkan secara horizontal, dari kanan ke kiri
    • flex-direction: column; elemen ditempatkan secara vertikal dari atas ke bawah
    • flex-direction: column-reverse; elemen ditempatkan secara vertikal dari bawah ke atas

    2. justify-content: mengatur penempatan elemen dalam container secara horizontal (kiri ke kanan)

    • justify-content: flex-start; elemen ditempatkan di awal/kiri container (nilai default pada sumbu utama)
    • justify-content: flex-end; elemen ditempatkan di akhir/kanan container pada sumbu utama
    • justify-content: center; elemen ditempatkan ditengah container secara horizontal
    • justify-content: space-between; elemen-elemen ditempatkan dengan jarak yang merata di antara mereka pada sumbu utama
    • justify-content: space-around; elemen-elemen ditempatkan dengan jarak yang merata di sekitar mereka pada sumbu utama

    3. align-items: mengatur penempatan elemen dalam container secara vertikal (atas ke bawah)

    • align-items: flex-start; elemen ditempatkan di atas container (nilai default)
    • align-items: flex-end; elemen ditempatkan di bagian bawah container
    • align-items: center; elemen ditempatkan di tengah container secara vertikal
    • align-items: baseline; elemen-elemen ditempatkan pada garis dasar yang sama/merata
    • align-items: stretch; elemen-elemen diperluas agar mencakup tinggi container

    4. flex-wrap: mengatur apakah elemen dalam kontainer flex akan melingkari (wrap) ke baris baru jika tidak cukup ruang

    • flex-wrap: nowrap; elemen-elemen tetap dalam satu baris (nilai default)
    • flex-wrap: wrap; elemen-elemen melingkari (wrap) ke baris baru jika tidak cukup ruang

    5. align-content: memiliki efek ketika elemen dalam container flex melingkari ke baris baru

    • align-content: flex-start; jika elemen dalam container melingkari garis baru maka elemen akan di peratakan di awal baris pertama
    • align-content: flex-end;
    • align-content: center;
    • align-content: space-between;
    • align-content: space-around;
