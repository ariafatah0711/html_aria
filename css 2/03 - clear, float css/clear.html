float clear

clear: left; clear: right; elemen tidak akan muncul di sebelah kiri atau kanan elemen yang memilki float
clear: both; elemen tidak akan muncul di kiri dan kanan dan akan berada di bawah

cara membersihkan/clear float

  • menggunakan propety overflow
    • container {overflow: auto;}
  • menggunakan <div> kosong
    •                 misal div 1 memilki float di kiri yang menyebabkan div 2 berada di samping div 1
                dan cara mengatasinya adalah dengan cara menabahkan div yag memilki class clear dan di class tersebut
                ditambahkan propety clear: [left]; maka elemen yang di atas div yang memilki efek float akan dihapus oleh div kosong tsb
  • menggunakan teknik micro clearfix
    •                 .cf:before, .cf:after {
                    content: " "; #ini adalah konten kosong(dapat berisitext juga)
                    display: table; 
                    clear: both;
                }
