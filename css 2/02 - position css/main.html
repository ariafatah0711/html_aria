position-static position-relativ position-absolute position-fixed position-sticky

position: static; selalu mengikuti posisi tag sebelumnya (nilai default)
position: relativ; elemen relativ terhadap posisinya yang berarti dapat menggunakan propeti top, right, left, bottom, dll. tetapi tidak mempengaruhi tata letak elemen di sekitarnya
position: absolute; elemen diposisikan relativ terhadap elemen induk yang berposisi tidak statis (bukan posisi default)/ elemen setelahnya akan dapat menempati posisi elemen
position: fixed; elemen tersebut akan ditempatkan relativ terhadap viewport (area tampilan pada layar), dan posisinya akan tetap tidak berubah saat pengguna menggulir halaman (mirip absolute namun dia akan tetap tampil meski di srcoll)
position: sticky; memungkinkan mengikuti alur dokumen mormal sampai batas tertentu, dan setelah itu akan berperilaku seperti elemen fixed hingga mencapai batas akhir
