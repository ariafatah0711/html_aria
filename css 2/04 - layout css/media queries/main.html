fixed width liquid responsive layout media queries

©media screen and (max-width:600px){ }
berfungsi sebagai css untuk yang memiliki media/device dengan maksimal 600px
jenis media ©media all{ }
media query akan berlaku untuk semua jenis media

©media screen{ }
media query akan berlaku ketika halaman ditampilkan pada perangkat layar monitor, ponsel, atau tablet

©media print{ }
media query akan berlaku ketika halaman dicetak

©media speech{ }
media query akan berlaku ketika halaman dibaca oleh teknologi pembaca layar atau proga, text-to-speech
propety media max-width: 500px; menentukan lebar maksimum elemen untuk menerapkan aturan css
min-width: 100px; menentukan lebar minimum
max-height: 1000px; menentukan tinggi maksimum elemen untuk menerapkan aturan css
max-height: 500px; menentukan tinggi minimun

orientasi: portrait; menentukan apakah perangkat memilki tampilan dalam mode potrait(vertikal) atau lanscape(horizontal)
orientasi: landscape; media query akan berlaku untuk tampilan landscape

min-resolution: 300dpi; media query akan berlaku untuk kepadatan pixel per inch atau dpi dengan resolusi tertentu
