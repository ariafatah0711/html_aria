<meta charset="UTF-8">>
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
meta digunakan untuk menyediakan informasi meta atau metadata tentang halaman web

1. charset="UTF-8" berfungsi untuk menentukan set character (tidak penting)
2. name dan content berfungsi untuk menyediakan meta data khusus (penting agar responsive layout)
name="description" content="deskripsi singkat halaman web" atau name="viewport" content="width=device-width, initial-scale=1.0"
3. http-equiv dan content berfungsi untuk meniru perilaku dari http header tertentu
http-equiv="X-UA-Compatible" content="IE=edge" atau http-equiv="refresh" content="10"
