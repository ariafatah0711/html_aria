selector css selector class css

  1. Selector Elemen
    2. <p> memilih elemen berdasarkan nama elemen misalnya memilih semua elemen p 
                        p {
                        color: red;
                    }
  2. Selector ID
    3. <div id="header"> memilih elemen berdasarkan nama id mereka misalnya #header 
                        #header {
                        color: red;
                    }
  3. Selector class
    4. <div class="body"> memilih elemen berdasarkan class mereka misalnya .body 
                        .body {
                        color: red;
                    }
  4. Selector potongan elemen
    5. <ul> <li> memilih elemen li di dalam elemen ul. missalnya ul li 
                        ul li {
                        color: red;
                    }
                    .header .header-title {
                        color: red;
                    }
  5. Selector anak langsung
    6. <ul> <li> memilih elemen li yang merupakan anak langsung dari elemen ul. misalnya ul > li 
                        ul > li {
                        color: red;
                    }
                    .header > .body {
                        color: red;
                    }
  6. Selector pesudo class
    7. <p> elemen memilih elemen berdasarkan statusnya, seperti saat di hover, di klik, atau difokuskan. misalnya p:hover 
                        p:hover {
                        color: red;
                    }
                    p:active {
                        color: red;
                    }
                    p:focus {
                        color: red;
                    }
  7. Selector pesudo elemen
    8. <p> memungkinkan kamu menambahkan konten sebelum atau setelah isi elemen. misalnya p::before 
                        p::before {
                        content: "<-";
                    }
                    p::after {
                        content: "->";
                    }
  8. selector bersaudara
    9. <h2> <p> memilih elemen p yang berada langsung setelah elemen h2 (semua elemen p yang ada di h2). misalnya h2 + p 
                        h2 + p {
                        color: red;
                    }
                    h2 - p {
                        color: red;
                    }
  9. selector universal
    10. semua elemen memilih semua elemen. misalnya * 
                        * {
                        margin: 0px;
                        box-sizing: border-box;
                    }
  10. selector :nth-child
    11. :nth-child(N) memilih elemen ke-N dalam hireki orang tua. misalnya ul li:nth-child(odd) 
                        ul lo:nth-child(odd) {
                        background-color: lightgrey;
                    }
                    ul lo:nth-child(even) {
                        background-color: lightblue;
                    }
                    ul lo:nth-child(3) {
                        background-color: red;
                    }
  11. selector elemen berdasarkan atribut
    12. input[type="text"] memilih elemen input dengan type text. misalnya input[type="text"] 
                        input[type="text" {
                        background-color: lightgrey;
                    }
                    img[src="/image/1.jpg"] {
                        background-color: lightblue;
                    }
