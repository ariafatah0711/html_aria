HTML - 02

Latihan HTML dan Style

Latihan ini dirancang khusus bagi pemula yang ingin mempraktikan pengetauan dasar tentang HTML (Hyper Markup Language). dalam latihan ini, anda akan membuat sebuah portofolio kecil/ website simple dengan style dasar

CSS - 01

Pengenalan CSS Dasar

Pengenalan CSS dasar melibatkan pemahaman tentang Cascading Style Sheets (CSS), yang digunakan untuk mengatur tampilan dan format halaman WEB, dalam CSS dasar, anda akan belajar tentang penggunaan selector untuk memilih elemen HTML dan aturan deklaratif

CSS - 02

CSS Layout

CSS layout melibatkan memahami bagaimana mengatur tampilan dan struktur elemen-elemen di halaman WEB menggunakan Cascading Style Sheet (CSS). saat anda mempelajari CSS layout, anda akan diperkenalkan dengan konsep dasar seperti tata letak Bock dan inline, serta mempelajari position

CSS - 03

CSS Lanjutan

pada tahap CSS lanjutan, anda akan mempelajari konsep-konsep lebih mendalam. ini termasuk penggunaan ::before dan ::after, serta mempelajari konsep Variable CSS('--variable') dan bagaimana memungkinkan anda memilih elemen berdasarkan nama kelas yang mengandung substring tertentu

CSS - 04

Latihan CSS

melalui latihan CSS, anda akan dapat menguji dan memperdalam pemahaman tentang konsep selektor, tata letak, pseudo-elemen, dan propeti khusus. latihan juga membantu anda menyelesaikan masalah dan mengembangkan kreativitas
