JQuery 
    
                - JQuery meerupakan libary javascript yang menyderhanakan fungsi javascript pada JQuery
                - sebelum menggunakan JQuery perlu menaruh script ini ke dalam head html
                    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.0/jquery.min.js"></script>

                - untuk mengambil data dapat menggunakan fungsi Load
                    // load data ke elemen tertentu via AJAX
                    $(Selector).load(URL, data, callback);

                    // ambil data dari server
                    $.get(URL, callback);

                    // kirim data dari server
                    $.post(URL, data, callback);

                - contoh penggunaan untuk menampilkan object api github
                    $("#nama_id").load("url");
                    $("#result").load("https://api.github.com/users/ariafatah0711");
                    maka id result akan menampilkan object dari api tsb

                    fungsi load() untuk mengambil bagian dari HTML untuk ditampilkan
mengambil data 
                - mengambil avatar_url, name, location menggunakan JQuery

                    var url = "https://api.github.com/users/ariafatah0711";
                    $.get(url, function(data, status) {
                        $("#avatar").attr("src", data.avatar_url);
                        $("#name").text(data.name);
                        $("#location").text(data.location);
                    });
mengirim data 
    
                - mengirim data title, body, userId

                    var url = "https://jsonplaceholder.typicode.com/posts";
                    var data = {
                        title: "Ini adalah contoh AJAX dengan JQuery",
                        body: "Ini adalah body",
                        userId: 1
                    };
            
                    $.post(url, data, function(data, status) {
                        // data terkirim, lakukan sesuatu di sini
                        console.log("data terkirim");
                    })
