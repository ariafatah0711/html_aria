menulis javascript di atribut 
                menulis javascript di atribut hanya bisa ditulis pada atribut event
                1. keybord event
                    - onkeydown="" = atribut akan dijalankan saat tombol ditekan
                    - onkeyup="" = terjadi saat tombol keyborad dilepaskan
                    - onkeypress="" = terjadi saat tombol keybord ditekan dan dilepaskan
                    - onfocus="" = terjadi saat elemen mendapatkan fokus misalnya saat pengguna mengklik elemen input
                    - onblur="" = terjadi saat elemen kehilangan fokus misaalnya setelah pengguna mengklik elemen input
                    
                2. mouse event
                    - oneclick="" = akan menjalankan script saat di klik
                    - ondbclick="" = terjadi saat elemen di klik dua kali dengan cepat
                    - onmousedown="" = terjadi saat tombol mouse utama di tekan di elemen
                    - onmouseup="" = terjadi saat tombol mouse utama dilepaskan setelah sebelumnya ditekan pada elemen
                    - onmouseover="" = terjadi saat kursor mouse bergerak di atas elemen

                3. other
                    - onsubmit="" = akan menjalankan fungsi saat elemen form di submit
                    - onchange="" = atribut di jalankan saat isi elemen input berubah misalnya pada elemen select
                    - onload="" = atribut ini dijalankan saat elemen atau halaman selesai dimuat
